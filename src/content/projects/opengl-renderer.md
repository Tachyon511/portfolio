## Overview
3D rendering engine wrote in C++ with the use OpenGl and GLSL, aimed as a learning project for low-level graphics-programming.

The current version supports:
- Diffuse & Specular Lighting & Materials
- Directional Lights & Shadows
- Point Lights & Omnidirectional Shadows
- Model importing and rendering
- Texture rendering using UV mapping
